BANCO DE DADOS:

BD - coleção organizada de dados
ex: lista telefônica
fonte de informação
são os dados dispostos de forma organizada

SGBD (Sistema de Gerenciamento de Banco de Dados) - 
aplicações que manipulam banco de dados,
funcionam como uma ponte entre o usuário, o cliente e a 
coleção de dados em si

SERVER - servidor, "servir", o servidor serve o SGBD
